## How to donwload?
run this commands on terminal
```
cd Desktop
git clone https://github.com/theiskaa/pokedex
```
then you have to run pub get command
```
flutter pub get
```
And then you can run the application
```
cd pokedex 
flutter run
```

## Package i use:
- [http](https://pub.dev/packages/http)
